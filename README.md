# Technical Assignment

The Recipe App is a React application that allows users to browse and manage recipes. Users can filter recipes by category, perform a search with debounce, and use pagination. The app provides a smooth user experience through frontend filtering and pagination, while the search is done via an API.

[DEMO LINK](https://nataliatopornitska.github.io/technical_assignment/)

## Technologies Utilized:
- React
- TypeScript
- Tanstack Query
- CSS3
- Fetch, REST API
- GitHub
